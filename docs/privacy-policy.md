# AARU Privacy Policy Draft

Last updated: March 9, 2026

## What AARU Collects

AARU stores the information needed to run the app:

- device-scoped session identifiers
- soul profile text and generated profile fields
- avatar selections
- conversation messages and compatibility results
- technical logs needed to diagnose service failures

## How AARU Uses Data

We use this data to:

- create and maintain your in-app identity
- generate profile and conversation experiences
- save your world state and message history
- keep the service stable and secure

## Sharing

AARU uses Supabase as its backend platform for data storage and server functions. We do not sell personal data.

## Retention

We retain data only as long as needed to operate the service, investigate abuse, and comply with legal obligations. Define the exact retention window before public release.

## Your Choices

Before App Store submission, AARU should provide:

- a support contact for privacy requests
- a process to delete account-linked or device-linked data
