How to export production screenshots

  • Option A (easiest): Open each screenshot-N link above in Chrome. Set window to exact size: Cmd+Opt+I → device toolbar → set to 1290x2796. Take a full-size screenshot (Cmd+Shift+P → "Capture full size screenshot").
  • Option B (automated): Run the included export.sh script which uses Chrome headless to render all 3 at exact pixel dimensions.
  • Required resolution: 1290 x 2796 px (iPhone 6.7"). Apple auto-scales for smaller devices from this size.
  • Panoramic chaining: The 3 screenshots share a continuous background gradient. When users swipe in the App Store, the background flows across panels seamlessly.
  • Glassy buttons: The "Begin" button and send button use iOS 26-style glass material — semi-transparent gradient with specular highlights, inner shadows, and backdrop blur.
